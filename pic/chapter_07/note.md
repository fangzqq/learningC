# 使用函数

将程序设计任务分解为函数，使得代码更容易编写，阅读， 理解， 调试， 修改和维护：

- 理解函数的基础知识
- 解释局部，全局，自动和静态变量
- 与函数一起使用一维和多维数组
- 从函数返回数据
- 用函数执行自上而下的程序设计
- 在函数中调用其他函数，以及递归函数

## 定义函数

函数定义的第一行向编译器告知函数的四件事：

- 谁可以调用它
- 它返回什么类型的值
- 他的名字
- 它接受什么参数

## 局部，全局，自动和静态变量

自动局部变量：在一个函数内部定义的变量称为自动局部变量

### 声明返回类型和参数类型

```c
float absoluteValue(float);

int printf(char *format, ...);
```

### 自上而下的程序设计

指定函数接受的参数和返回值即可，不用马上考虑函数的实现细节，然后就可以编写对这一函数的调用。
