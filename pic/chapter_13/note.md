## 枚举数据类型

枚举数据类型可以让我们定义一个变量，并指定可以在该变量中存储的有效值。

```c
enum primaryColor { red, yellow, blue};
```

c 编译器实际上将枚举标识符看作整型常数，从列表中的第一个名字开始，编译器依次为这些名字分配起始值为 0 的连续整数值。

```c
enum direction {up, down, left = 10, right};
// up: 0, down: 1, left: 10, right: 11
```
